LLM Boids

Interactive crowd simulation system combining Reynolds' Boids algorithm with RBF flow fields and natural language control. Extended an existing 3D RBF field work by implementing an LLM-driven control layer that parses natural-language commands and generates structured JSON directives to adjust Boid parameters and flow-field source vectors in real-time within Unity. Features a visual debug interface and scene-aware prompting for intuitive feedback and control, enabling designers to author complex agent behaviors without manual parameter tuning.

C# Unity LLM Crowd Simulation Boids RBF Fields Natural Language Processing

Digosaur

Interactive museum installation at Tekniska Museet (Swedish Museum of Technology) using the Samsung SUR40 table. Children can search and dig for dinosaur bones to complete a full skeleton through intuitive touch interactions. Developed a custom input processing library for touch data, implemented TCP network communication between the SUR40 table and Godot game engine, and created real-time particle effects that follow hand movements to simulate digging through sand.

C# Godot TCP Input Processing Network Programming Interactive Installation
